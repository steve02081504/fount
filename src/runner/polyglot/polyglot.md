# Polyglot

此项目支持以下运行方式：

## 运行

- `sh ./run.sh`
- `cmd /c .\run.bat`
- `sh ./path/fount`
- `bash ./path/fount.sh`
- `cmd /c .\path\fount.bat`
- `powershell ./path/fount.ps1`
- `pwsh ./path/fount.ps1`
- `deno run -A ./path/fount.mjs`
- `node ./path/fount.mjs`
- `sh ./run.bat`
- `sh ./path/fount.bat`
- `sh ./path/fount.ps1`
- `sh ./path/fount.mjs`
- `make run`
- `npm start`
- `deno task start`
- `cargo run`
- `dotnet run`
- `gradle run`
- `./gradlew run`
- `swift run`
- `task run`
- `just run`
- `gulp`
- `mix run`
- `stack run`
- `sbt run`
- `docker compose up`

## 安装

- 任何运行代码会自动安装后再运行
- 任何`fount init`
- `make install`

## 卸载

- 任何`fount remove`
- `make uninstall`
